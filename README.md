dotfiles
========

My dotfiles repos.

## Installation
**当リポジトリの install.sh は OS X で動作します。**

1. Xcode をインストール
2. Xcode Command Line Tools をインストール
```
xcode-select --install
```
3. リポジトリをクローン
```
git clone https://github.com/moltpw/dotfiles.git ~/dotfiles
```
4. インストールスクリプトを実行
```
cd ~/dotfiles
./install.sh
```
